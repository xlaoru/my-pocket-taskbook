### Білд Docker

<ul>
<li><code>docker-compose down</code> -- зупинка Докеру</li>
<li><code>docker-compose up -d</code> -- запуск Докеру</li>
</ul>

### Взаємодія з PostgreSQL через Docker

<ul>
<li><code>docker exec -it my_pocket_taskbook psql -U illia -d db</code> -- запуск psql</li>
<li><code>\dt</code> -- перегляд таблиці</li>
<li><code>\d TABLE_NAME</code> -- перегляд типів конкретної таблиці таблиці</li>
<li><code>\q</code> -- вихід з psql</li>
</ul>
